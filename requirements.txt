# Runtime dependencies for cognitive-discovery-system.
#
# The source of truth is pyproject.toml ([project.dependencies]). This file is a
# convenience mirror for environments/workflows that expect a requirements.txt
# (e.g. some CI images, Binder, plain virtualenvs). Keep the two in sync.
#
# Install:  pip install -r requirements.txt
#
# The package is zero-dependency at runtime: the computational core and the
# `cds` CLI are implemented purely on the Python standard library (argparse,
# dataclasses). Previously typer/pydantic/rich were required, but the v1.1.9
# refactor removed them entirely, so this file is intentionally empty — there
# is nothing to install beyond Python itself.
